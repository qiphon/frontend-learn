米多多是什么？
米多多是一款为金融从业人员提供快捷金融产品的分销平台，零门槛，完成注册即可代理各类金融产品。销客赚平台提供：贷款、信用卡、征信等多元化金融产品。
平台还为用户提供订单查询、客户管理、佣金奖励查询等功能，以及多种营销工具让用户业务开展和后期维护更加方便。
获得佣金奖励的方式有哪些？
直销佣金：通过分享贷款、信用卡、征信等金融产品发展的客户，放款成功、核卡成功或投资成功后平台按照一定比例或固定金额进行返佣；
徒弟奖励：用户邀请好友成为平台会员，该好友获得了直销佣金后，平台按照好友直销佣金的一定比例进行返佣；
徒孙奖励：用户邀请好友成为平台会员，该好友又邀请了其他好友成为平台会员，其他好友获得了直销佣金后，平台按照其他好友直销佣金的一定比例进行返佣；
如何发展团队？
发展团队就是收徒弟，再让徒弟去发展徒孙，徒弟和徒孙都是您的团队成员。
通过“首页—我要收徒”功能，将分享海报或者邀请链接分享至微信好友、朋友圈、微博等，好友成功注册后成为分享人的徒弟；徒弟再邀请其他好友注册成功，其他好友将成为分享人徒孙。
发展团队的好处是什么？
徒弟和徒孙开展贷款、信用卡、理财等业务，他们收到佣金奖励后，销客赚平台都将按照一定比例额外给您奖金。
徒弟和徒孙的关系终身有效并且没有数量限制，发展徒弟越多，徒孙就越多，同样收获的奖金就越多。
